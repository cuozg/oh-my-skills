Load the `unity/unity-investigate` skill and perform a deep investigation.

## Task

$ARGUMENTS

## Investigation Scope

Analyze thoroughly, covering all applicable areas:
- **Logic flow** - Trace execution from trigger to outcome
- **Data structures** - Understand data models and serialization
- **Resource management** - Asset loading, memory, lifecycle
- **System interactions** - How components communicate
- **Side effects** - What else gets triggered
- **Performance** - Identify bottlenecks or inefficiencies

## Output

Provide a comprehensive analysis with:
1. Architecture overview of the investigated system
2. Execution flow with key entry points
3. Data flow and dependencies
4. Potential issues or improvement opportunities
5. Code references with file paths and line numbers
