## Documentation Types

Generate the appropriate documentation type:
- **README** - Project overview, setup, quick start
- **Architecture** - System diagrams, component relationships, data flow
- **API Reference** - Public interfaces, method signatures, usage examples
- **Onboarding Guide** - Step-by-step setup for new developers
- **Prefab/Scene Setup** - Inspector configuration, hierarchy structure

## Requirements

1. Investigate the codebase thoroughly before writing
2. Include code examples where relevant
3. Use clear headings and formatting
4. Keep documentation maintainable and up-to-date
