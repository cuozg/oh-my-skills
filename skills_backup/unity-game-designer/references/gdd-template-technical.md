# Game Design Document - Technical & Roadmap

## 9. Technical Constraints & Scope

```markdown
### 9.1 Engine & Platform
- Engine: Unity 6+
- Platforms: iOS 14+, Android 11+
- Target Performance: 60fps on mid-range devices

### 9.2 Design Constraints
- Max level cap: 200 (prevents long-tail balance issues)
- Max inventory: 999 per item type
- Session length target: 5-15 minutes
```

## 10. Launch & Long-Term Roadmap

```markdown
### 10.1 Launch Milestones
- **Alpha**: Core loop, shop, 5 progression tiers
- **Beta**: FTUE, monetization, leaderboards
- **1.0**: Full launch, 10 seasonal events planned

### 10.2 Post-Launch Plan (Seasons 1-4)
- **S1**: New weapon class + 3 cosmetics
- **S2**: Guild wars + 5 cosmetics
- **S3**: PvP ranking + battle pass tier 2
- **S4**: New game mode + cosmetic rework

### 10.3 Failure Metrics (When to Pivot)
- If D1 retention < 30% → FTUE broken, redesign tutorial
- If LTV/CAC < 1.5 → Economy broken, reduce prices
- If daily active < target by week 4 → Core loop too grindy
```
