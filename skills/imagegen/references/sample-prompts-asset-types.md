# Sample prompts — Asset type templates

Recipes for common asset types. Keep user requirements; do not invent elements.
Principles: `references/prompting.md` · Taxonomy: `references/prompt-augmentation.md`

## Website assets

Template: hero, section illustration, blog header.
```
Use case: <photorealistic-natural|stylized-concept|product-mockup|infographic>
Asset type: <hero image / section illustration / blog header>
Primary request: <short description>
Style/medium: <photo/illustration/3D>
Composition/framing: <wide/centered; negative space side>
Color palette: <brand colors or neutral>
Constraints: <no text; no logos; no watermark>
```

Hero: minimal abstract background + soft gradient. Flat illustration. Cool neutrals + blue accent.

Feature section: abstract shapes suggesting connection. Flat illustration. Muted teal + slate. Centered cluster.

Blog header: overhead desk scene (notebook, pen, coffee). Photorealistic. Soft morning light. Subject left; right empty.

## Game assets

Environment: cavernous hangar. Wide-angle, low-angle. Volumetric light rays through fog.

Character: desert scout. Long coat, hood, boots. Stylized realism. Neutral hero pose.

UI icon: round shield with rune. Painted style. Centered. Transparent background.

Texture: worn sandstone blocks. Seamless tileable. Neutral lighting. No focal elements.

## Logo concepts

Geometric leaf: growth + sustainability. Vector mark. Deep green + off-white.

Monogram: interlocking letters (e.g., "AV"). Vector. Black on white.

Wordmark: clean text mark. Charcoal on white. Even letter spacing.

## Product mockups

Bottle/jar: premium product. Studio gradient background. Softbox lighting. Matte plastic. Centered. 3/4 angle.

Packaging: product box or pouch. Studio background. Clean lighting. Label legible. No watermark.

Merch concept: T-shirt, hoodie, or hat. On-person or flat-lay. Clean staging. Colors accurate.

## Infographics

Flow diagram: process steps (e.g., coffee machine flow). Vertical layout. Clear arrows + labels. Vector-like.

Timeline: events/milestones along horizontal line. Icons + text callouts. Clean color palette.

Comparison chart: side-by-side features or specs. Table/columns. Clear typography. No logos.

## Editorial & lifestyle

Magazine cover: bold headline area. Hero image focal. Masthead top. Clean typography. Magazine-style.

Social media: square 1:1 or 16:9. Bold headline. Visual anchor. Brand colors. Safe text area margins.

Poster: event or campaign. Bold composition. Clear hierarchy. Key info legible. Print-ready colors.

## UI mockups

App home: app name top. Cards/sections below. Practical hierarchy. iPhone frame. No logos.

Dashboard: summary cards. Data tables. Charts. Light neutral background. Clear info grouping.

Form: input fields labeled. Buttons clear. Error states visible. Clean white space.
